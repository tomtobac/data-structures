## ⏲️ Time Complexity (Big O Notation)

|                |        |
| -------------- | ------ |
| linear time    | O(n)   |
| constant time  | O(1)   |
| quadratic time | O(n^2) |

How to calculate it:

1. find the fastes growing term
2. take out the coeficient

> T = an + b = O(n);

> T = cn^2 + dn + e = O(n^2)
